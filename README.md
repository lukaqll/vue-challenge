## Instalação

Instale vue-challenge com npm

```bash
  git clone https://github.com/lukaqll/vue-challenge.git
  cd vue-challenge
```
    
## Deploy

Para fazer o deploy desse projeto rode

```bash
  npm install
```

```bash
  npm run build
```

```bash
  cd dist && npx serve -l 3050
```

abra a rota http://localhost:3050 em seu nvegador
